# Reported Thesis Results

This folder contains reported-result summaries transcribed from thesis document and presentation materials reviewed during private extraction.

These are reported results, not independently reproduced results. They have not been verified against standalone raw solver outputs, rerun logs, or exported computational artifacts.

The raw thesis DOCX/PPTX files are intentionally not included in this public branch.

No standalone `.lst`, `.gdx`, solver logs, raw CPLEX/GAMS outputs, or exported result tables are currently included in this repository.

Use these files as a cautious index of what is reported in the thesis/presentation. Any future documentation should distinguish these reported values from independently reproducible results.
